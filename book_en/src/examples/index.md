# الأمثلة

هذا المشروع يحتوي مجموعة أمثلة واسعة. شغّل أي مثال بصيغة:

```bash
cargo run --release --example <name>
```

## أمثلة أساسية

- `hello_world`: البداية السريعة.
- `text_label`: سلوك النص.
- `texture`: الصور/الخامات.
- `widgets`: تجميعة widgets عامة.
- `interaction`: حالات التفاعل.

## أمثلة Widgets

- `toggle`
- `radio`
- `seekbar`
- `drag_value`
- `select`
- `text_field`
- `scroll_view`
- `panel_divider`
- `panel_window`
- `card_profile`

## أمثلة أخرى

- `alignment`
- `masonry`
- `layout_cache`
- `ex_node`
- `border_light_3d`
- `sci_fi`

## توصية عملية للتعلم

1. ابدأ بـ `hello_world`.
2. انتقل إلى `widgets` و`interaction`.
3. ثم راجع `layout_case_*` حسب النمط المطلوب.
4. أخيرًا افحص `sci_fi` كحالة مركبة كبيرة.
