# مرجع الـ API

هذا القسم يقدّم فهرسًا عمليًا لسطح الـ API العام في `univis_ui`.

## أين تجد المرجع الآلي الكامل؟

لتوليد rustdoc:

```bash
cargo doc --no-deps
```

ثم افتح:

```bash
cargo doc --no-deps --open
```

## كيف تستخدم هذا القسم؟

- ابدأ بـ [الأحداث Messages](events.md) إذا كنت تربط UI مع gameplay/app logic.
- راجع [جدول الأنواع العامة](public-types.md) للوصول السريع لكل struct/enum/plugin.
